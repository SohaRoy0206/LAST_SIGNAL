# LAST SIGNAL 🚀

An AI-powered predictive healthcare system for satellites that detects early warning signs of failure using telemetry analysis, anomaly detection, and forecasting models.

**Last Signal** helps prevent catastrophic satellite breakdowns through intelligent monitoring and real-time risk prediction.

## Project Overview

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

### Tech Stack
- **Framework**: Next.js 16.2.6 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Bundler**: Turbopack
- **Linting**: ESLint

### Features
- Futuristic satellite monitoring dashboard
- Real-time satellite health metrics
- Failure risk assessment
- Signal status monitoring
- NASA mission control-inspired cyberpunk UI design
- Glassmorphism cards with animated effects
- Fully responsive design

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the LAST SIGNAL dashboard.

The dashboard displays:
- **Satellite Health**: Real-time operational status (92%)
- **Failure Risk**: Current failure prediction (8%)
- **Signal Status**: Connection stability (Stable)
